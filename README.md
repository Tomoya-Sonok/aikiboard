# AikiBoard

AikiBoard（アイキボード）は、合気道の道場・会などのコミュニティを運営する管理者向けの **道場管理＆コミュニケーションプラットフォーム** です。
個人向け稽古日誌アプリ **AikiNote** の姉妹サービスとして、共通アカウント基盤のもとで連携しながら提供される想定です。

> **⚠️ 本プロジェクトは準備段階です。**
>
> 要件定義・技術選定はまだ完了していません。詳細は今後の慎重な要件定義プロセスを通して FIX していきます。
